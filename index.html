Apa Itu Wordpress

WordPress adalah platform yang mudah digunakan untuk pemula dan terbukti sudah digunakan lebih dari 455 juta website di seluruh dunia.

Artinya, kalau Anda ingin membuat website, apapun jenisnya, WordPress bisa jadi pilihan terbaik. Nah, kalau Anda ingin mengenal WordPress sebelum memutuskan untuk menggunakannya, inilah artikel yang tepat untuk Anda.

Install

cd /tmp && wget https://wordpress.org/latest.tar.gz

tar -xvf latest.tar.gz

cp -R wordpress /var/www/html/

chown -R www-data:www-data /var/www/html/wordpress/

chmod -R 755 /var/www/html/wordpress/

mkdir /var/www/html/wordpress/wp-content/uploads

chown -R www-data:www-data /var/www/html/wordpress/wp-content/uploads/
